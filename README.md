# 介绍
- 每当运行时，lsp会从[图库](lsp-db)随机抽取一张图片
- 本图库的大部分图片来源于网络, 部分来自于闲暇时收集的图片
- 一天只可抽取一张色图，不能抽多了，看多了伤身体:p 🤣
- 看完的图片会被立刻删除，因为我也不知道为啥哈哈哈
- 所有图片都已经过严格筛选，请大家谨慎使用！
